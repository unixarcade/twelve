# Twelve 🕯️

**A quiet toolkit for recovery. One day at a time.**

This is a free, simple web app built around the 12 Steps and 12 Traditions — with small, practical tools woven into each one. A breathing exercise. A place to write down resentments. A God Box where you can type a worry and let it go. A spot check at the end of the day.

It was made with love, for people who are fighting hard.

---

## For My Mom

This app was built in memory of my mother, who struggled with the disease of addiction.

She is the reason this exists. She is why it's free. She is why it tries to be gentle.

## What It Is

Twelve is a single HTML file. No login. No account. No ads. No data collected. Nothing sent anywhere.

Every note you write, every list you build, every name you add to your amends — it stays on your device and nowhere else. Your story is yours.

Each of the 12 Steps opens a small tool:

| Step | Theme | Tool |
|------|-------|------|
| 1 | Honesty | Powerlessness journal |
| 2 | Hope | Emergency sanity anchors |
| 3 | Surrender | The God Box — type a worry, release it |
| 4 | Courage | Resentment list builder |
| 5 | Integrity | Speaking truth prompt |
| 6 | Willingness | Character drop toggles |
| 7 | Humility | The Humility Prayer |
| 8 | Love | Amends planner |
| 9 | Discipline | Action tracker |
| 10 | Perseverance | Nightly spot check |
| 11 | Spirituality | Guided breathing meditation |
| 12 | Service | Find meetings · Volunteer |

## For Developers

There is nothing to install. No `npm`. No build step. No dependencies on your machine.

The entire app is one file: `twelve.html`

It loads React, Tailwind, and Babel from public CDNs on first open. After that it works offline. If you want to make changes, open the file in any text editor and edit it directly.

To deploy your own copy on GitHub Pages:

1. Fork or clone this repo
2. Rename `twelve.html` to `index.html` if you want a clean URL
3. Go to **Settings → Pages → Deploy from branch → main**
4. Your app will be live at `https://yourusername.github.io/twelve`

## A Note on Data & Privacy

This app stores nothing on any server. The lists and journals you write live only in your browser's `localStorage` — on your own device, like a private notebook. Clearing your browser data will clear them. Nothing is ever transmitted anywhere.
